Welcome to flask-restplus-udata's documentation!
==========================================

flask-restplus-udata is an extension for Flask that adds support for quickly building REST APIs.
flask-restplus-udata encourages best practices with minimal setup.
If you are familiar with Flask, flask-restplus-udata should be easy to pick up.
It provides a coherent collection of decorators and tools to describe your API
and expose its documentation properly (using Swagger).


Compatibility
=============

flask-restplus-udata requires Python 2.7+.


Installation
============

You can install flask-restplus-udata with pip:

.. code-block:: console

    $ pip install flask-restplus-udata

or with easy_install:

.. code-block:: console

    $ easy_install flask-restplus-udata
